# TP_HW3
## Описание проекта
В проекте есть два разных Docker-контейнера (generator и reporter). Шаблонный скрипт под создание CSV-файла с данными был изменен для тематики авиабилетов (номер рейса, направление, авиакомпания, цена билета). Для каждой колонки изменены генераторы значений. 
## Контейнер generator
Запускает шаблонный скрипт generator.py, в результате работы создает файл data/data.csv
Docker-образ собирается на основе Python
## Контейнер reporter
Запускает шаблонный скрипт report.js, в результате работы создает файл data/report.html
Docker-образ работает на основе Node.js
